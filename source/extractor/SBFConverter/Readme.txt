# Licencing information

The SBFConverter uses the python module pcbnew from KiCad version 6. KiCad is distributed unter the GPLv3 license and
therefore, the SBFConverter inherits the same licence. The GPLv3 license file is added to this folder.

# Usage of SBF Converter

The SBFConverter is called "out of process" from the main programm (extractor) using the python.exe distributed
with KiCad. Therefore the extractor does not inherit the GPLv3 license.
